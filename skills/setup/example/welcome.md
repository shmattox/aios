# What is a review-gated pipeline?

A review-gated pipeline captures raw notes, drafts distilled knowledge from them,
then holds every draft at a gate before it becomes durable truth. Low-risk,
reconstructable items may auto-ship; anything touching money, ownership, or legal
terms is held for a human to approve. The gate is the point: the system proposes,
a person disposes, and every shipped change is revertible.

This example file exists so a fresh install has one deterministic capture to run
through ingest and the gate during setup's smoke test. Delete it once you have seen
your first staged draft.
